# Metapals Frontend Interview Test

By Matthew Christopher Albert

## Using

- Vanilla JS
- GSAP 3.x
- HTML2Canvas
- Tailwind 3.1.x
- SASS
- Vite 3.0

## Features

- Change Logo Text
- Change Logo Color
- Change Logo Position (Draggable)
- Convert to Canvas

## Deployment

- Github Pages

## Getting Started

```sh
yarn install
yarn dev # Start dev server of the project
# or
yarn build # Builds the project
```
